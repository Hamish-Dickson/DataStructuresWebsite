Overview of Lists

Put Simply, lists are a collection of 1 or more item, each with their own reference point, or index within the list.

They are useful for storing multiple data entries of the same data type, such as a list of phone numbers.

Example

Say we had a data set, perhaps lottery numbers for a person.

The numbers are 23, 43, 34, 18, 10 and 7.

If we were to use a list to store these items, we would have our data, with each item indexed from 0 through 5 (indexes start at 0. unless you are using some strange language, but mostly it's 0.)
