Array

An Array is a simple data structure that consists of a collection of elements, much like a list, wherein each element is identified by and index.

Arrays can have multiple dimensions, but we will be focusing on the basics, a one-dimensional array.

Example

Say we had a data set, perhaps lottery numbers for a person.

The numbers are 23, 43, 34, 18, 10 and 7.

If we were to use an array to store these items, we would have our data, with each item indexed from 0 through 5 (arrays start at 0. unless you are using some strange language, but mostly it's 0.)
